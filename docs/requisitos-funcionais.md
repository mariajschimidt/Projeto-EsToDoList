# Requisitos Funcionais - Gerenciador de Tarefas
1. [X]RF01 - O sistema deve permitir a adição de uma nova tarefa.
2. [X]RF02 - O sistema deve listar as tarefas existentes.
2. [X]RF03 - O sistema deve permitir a edição de uma tarefa existente.
3. [X]RF04 - O sistema deve permitir a exclusão de uma tarefa.
4. [X]RF05 - O sistema deve permitir marcar uma tarefa como concluída.
5. [X]RF06 - O sistema deve permitir a pesquisa de tarefas.
6. [X]RF07 - O sistema deve permitir a filtragem de tarefas.
